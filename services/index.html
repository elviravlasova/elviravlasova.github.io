Креативные идеи для требовательных клиентов

1. Подготовка

- сбор всей информации о будущем дизайн-проекте
- обсуждение бюджета, расчёт стоимости дизайн-проекта
- подписание договора на разработку дизайн-проекта
- составление четкого ТЗ для выполнения дизайн-проекта

2. Планировка

- стилистика проекта
- обмерный план
- фотографирование объекта
- планировочное решение (план возводимых перегородок, расстановку мебели, освещение)

3. Экскизный проект / Визуализация

- коллажи с концептом каждого пространства - предметы мебели, цветовые решения, материалы
- выбор всех отделочных материалов( обои, плитка, краска, декоративные покрытия)
- выбор светильников
- 3D визуализация

4. Рабочий проект

- альбом чертежей для проведения ремонтных работ
- развёртки стен помещений
- составление ведомостей, подсчёт необходимого количества материала
